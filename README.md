# 🍔 Black Fire Burgers

Site institucional de uma hamburgueria com cardápio digital, integração com WhatsApp e foco em conversão de vendas.

## 🚀 Funcionalidades

- Cardápio digital organizado
- Botão de WhatsApp integrado
- Promoção da semana em destaque
- Layout responsivo (mobile e desktop)
- Design focado em conversão

## 🛠️ Tecnologias

- HTML5
- CSS3

## 📱 Responsividade

O site foi desenvolvido para funcionar em dispositivos móveis, tablets e desktops.

## 💡 Aprendizados

- Estruturação de layout com Flexbox
- Criação de interface focada em vendas
- Organização de código HTML e CSS
- Boas práticas de responsividade

## 📌 Como executar

1. Baixe o projeto
2. Abra o arquivo `index.html` no navegador
